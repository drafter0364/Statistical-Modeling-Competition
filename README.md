# Statistical-Modeling-Competition

此为2024年全国大学生统计建模大赛1298号队伍的公开仓库。

以下为本仓库目录：

```
├─code
│      comment.py		# 爬取原始评论数据程序
│      dataclean.py		#数据清洗
│      epoch_loss.py		#绘制训练Loss-Epoch图
│      factor.py		#获取评论中的因素得分
│      parse.py			#分词
│      sentiment.py		#情感分析
│      tfidf.py			#计算TF-IDF
│
├─data
│      原始爬虫数据.zip
│      清洗后数据.zip
│
└─result
        TF-IDF.zip
        主题分析结果.zip
        好评语义网络图片.zip
        差中评语义网络图片.zip
        情感分析数据.zip
        训练损失图.zip
        贝叶斯网络文件.zip
```
